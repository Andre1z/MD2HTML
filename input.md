# Sample Markdown Document

This is a **demonstration** of Markdown syntax.  

## Headings
You can create headings with `#` (H1) to `######` (H6).  

## Text Formatting  
- **Bold** (`**text**` or `__text__`)  
- *Italic* (`*text*` or `_text_`)  
- ~~Strikethrough~~ (`~~text~~`)  
- `Inline code` (backticks)  

## Lists  
### Unordered List  
- Item 1  
- Item 2  
  - Nested item  

### Ordered List  
1. First step  
2. Second step  
   - Sub-item
